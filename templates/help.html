Help

Help documentation and documentation requirements

Blast
1.Select BLAST mode to upload database construction files and comparison files respectively.
2.The three parameters can be set by yourself.
3.The sample file is as follows.
protein   nucleic_acid  
Diamond
1.This feature is roughly the same as Blast, except that Diamond runs faster.
2.The sample file is as follows.
protein   nucleic_acid  
sequence_screening
1.The ID file is the ID file in the BLAST or Diamond result compression package. The user can upload CDS, PEP and GFF files of the species and extract the corresponding sequence according to the ID file.
2.The sample file is as follows.
CDS_file   PEP_file   GFF_file   ID_file  
Hmmer
1.HMMER is a bioinformatics software that analyzes gene sequences and can quickly determine how similar two sequences are.
2.The sample file is as follows.
model_comparison_file   Sequence_database_file  
Pfam
1.Pfam is a database of protein families, which are classified into different families based on multiple sequence alignment results and hidden Markov models.
2.The sample file is as follows.
pfam_file  
Meme_suit
1.The most commonly used tool for gene family analysis or other conserved sequence regions (motif or pattern) that need to display a set of sequences simultaneously is MEME suite.
2.The sample file is as follows.
meme_suit_file
Codonw
1.Codon bias analysis tool.
2.The sample file is as follows.
codonw_file
Cpgfinder
1.CpG islands are defined as stretches of DNA 500–1500 bp long with a CG: GC ratio of more than 0.6, and they are normally found at promoters and contain the 5′ end of the transcript.
2.The sample file is as follows.
Cpgfinder_file
GSDS
1.GSDS is an online service providing an all-English interface for displaying the composition and location of gene exons, introns, and conserved elements.
sequencerserver
1.SequenceServer is extensively used in private and public sector labs for cloning, comparative genomics, gene curation, design of diagnostic markers and many other applications. Our software has helped research on humans, viruses, bacteria, agricultural plants and animals, and dozens of emerging model organisms.

Cd_hit
1.Cd-hit is a tool for protein or nucleic acid sequence clustering. Sequences are clustered according to their similarity to remove redundant sequences. It is generally used to construct non-redundant data sets for subsequent experimental analysis.

Sequence_Alignment
1.Sequence alignment refers to the alignment of DNA,RNA, or protein sequences according to similarity, in which similar bases/amino acids are aligned in the same column by inserting gaps in the sequences.
2.Three multi-sequence tools are available.
3.The sample file is as follows.
input_protein_file  
MCscanX
1.It is convenient for users to find collinearity and synteny relationship, and clear multiple alignment can be seen from collinearity block.
2.The sample file is as follows.
blast_result_file   species_gff_file  
Colinearscan
1.ColinearScan is mainly based on dynamic programming algorithm and genetic collinearity method to detect homologous chromosome fragments.
2.The sample file is as follows.
target_database   FASTA_sequence1   FASTA_sequence2  
Orthomcl
1.ColinearScan is mainly based on dynamic programming algorithm and genetic collinearity method to detect homologous chromosome fragments.
2.The sample file is as follows.
target_database   FASTA_sequence1   FASTA_sequence2  
Orthofinder
1.Important functions include finding orthogonal groups and lineal homologues, inferring the root cause trees of all orthogonal groups, and identifying all gene duplication events in those gene trees. It also extrapolates the rooted species tree for the species it analyses and matches gene duplication events from the gene tree to branches of the species tree.
2.All four sample files are upload files, uploading multiple files at once.
Mycoplasma_agalactiae   Mycoplasma_gallisepticum   Mycoplasma_genitalium   Mycoplasma_hyopneumoniae  
ParaAT
1.Compared with clustal+ other scripting methods, this software has a high degree of automation. It automatically compares proteins first and then translates the protein sequence results back into corresponding nucleic acids, realizing parallel comparison of multiple groups of DNA sequences of homologous coding proteins.
2.The sample file is as follows.
homologs_file   cds_file   pep_file  
Kaks_calculator
